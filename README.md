## 🛠️ Getting Started
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/moolmin/chi-meong.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm run dev
   ```
