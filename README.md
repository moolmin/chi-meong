게임 요소를 더해 제주어를 친숙하게 인식하게 하고자 만들었어요.</br>

## 🛠️ Getting Started
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/moolmin/chi-meong.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm run dev
   ```
